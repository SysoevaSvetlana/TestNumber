# Number Sorting Microservice

Микросервис на Java с REST API для сохранения чисел в базу данных и получения отсортированного списка.

## Технологии

- Java 17
- Spring Boot 3.5.7
- Spring Data JPA
- PostgreSQL
- Liquibase
- Docker & Docker Compose
- TestContainers
- JUnit 5 & Mockito
- Swagger/OpenAPI 3.0

## Функциональность

REST эндпоинт принимает число, сохраняет его в БД и возвращает отсортированный список всех чисел.

# Быстрый старт

## Предварительные требования

- Docker Desktop должен быть установлен и запущен
- Порты 8080 и 5432 должны быть свободны

## Запуск приложения

### 1. Запустите Docker Desktop

Убедитесь, что Docker Desktop запущен и работает.

### 2. Запустите приложение 


Дождитесь сообщений:
- `Started TestApplication in X seconds` - приложение запущено
- Liquibase применил миграции

### 3. Проверьте работу


Откройте в браузере: **http://localhost:8080/swagger-ui.html**

1. Найдите эндпоинт `POST /api/numbers`
2. Нажмите "Try it out"
3. Введите число в поле (например, `3`)
4. Нажмите "Execute"
5. Увидите ответ с отсортированным списком
